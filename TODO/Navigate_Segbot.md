# Start segbot
Open a new terminal, and input

```
roslaunch segbot_bringup segbot_bu.launch
```

# Move segbot using joystick
Double check that the joystick is in D mode

```
roslaunch bwi_joystick_teleop joystick_teleop.launch
```
